## 💻 Getting Started <a name="getting-started"></a>


To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

```
 python installed
```

```
 Any code editor (VS Code)
```


### Setup

Clone this repository to your desired folder:

```
  git clone https://github.com/Theophileaseh/django-api-endpoint.git

  cd django-api-endpoint
```


### Install

Install this project with:

Run
```
  python -m venv .venv
```
to create virtual environmnet


``` pip install ```

Run ``` . .venv/Scripts/activate ``` to activate virtual environment

Run ``` python manage.py migrate ``` to run migrations


### Usage

To run the project, execute the following command:


```
  python manage.py runserver
```

Access api json data through ```[your server url]/doctors.json ```

Access single data through ```[your server url]/doctors/[doctor_id] ```

Access html view data through ``` [your server url]/doctors ```
